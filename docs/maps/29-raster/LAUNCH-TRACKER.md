## Notes & Observations

**Common Issues:**
- Pan/zoom issues with rain raster staying locked to map (most platforms initially)
- Color scheme visibility (dark rain on dark map)
- Animation playback choppiness
- Most issues fixed with follow-up prompts

**Platform Strengths:**
- **AntiGravity**: Good for larger projects, browser control, downloadable IDE
- **Gemini Canvas**: Almost worked first try, good iteration
- **v0**: Now using Opus 4.5, improved results

**Platform Weaknesses:**
- **ChatGPT Canvas**: Map/data rendering issues persisted
- **Claude Artifact**: Raster positioning problems

**Surprising Results:**
- **AntiGravity discovery**: New tool worked well, similar to Gemini 3 + AI Studio but better for larger projects
- **v0 improvement**: Opus 4.5 upgrade seems to help
- **Gemini Canvas**: Strong first attempt

- Downloadable Mac app, works outside browser
- Good for larger projects with more control

- Similar results to Gemini 3 + Google AI Studio
- Spent most development time here adding features
