## Prevention Checklist

Before launching prompts across platforms, check:

- [ ] Data size is reasonable (<1000 points for initial tests)
- [ ] Data format is standard (GeoJSON or simple JSON)
- [ ] Libraries are specified clearly
- [ ] Requirements are explicit, not implied
- [ ] One clear goal per prompt (not multiple tasks)
- [ ] Visual examples or descriptions provided
- [ ] Fallback options mentioned
- [ ] Mobile responsiveness requested
